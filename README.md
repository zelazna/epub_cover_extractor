# EpubCoverExtractor

A tool for get the ebooks covers, the projet is WIP,
contributions are welcome.

## Installation

Add `epub_cover_extractor` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:epub_cover_extractor, "~> 0.1.0"}
  ]
end
```
